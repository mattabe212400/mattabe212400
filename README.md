## What I'm Building

### OpsCore — Operations Management Platform
*Role-based operations platform built to solve a real organizational problem*

As Vice President of a 160+ member fraternity, I kept running into the same issues — attendance lived in spreadsheets, recruitment information was scattered across different tools, officer transitions depended on whoever happened to remember something, and there was no centralized way to manage operations.

Instead of accepting that as "just how things are," I decided to build a solution.

OpsCore is a full-stack operations platform that serves as a single source of truth for chapter management. It's currently deployed and being used by our organization, making it the most rewarding project I've worked on so far.

**Highlights**
- Role-based permissions across 20+ operational modules
- Real-time attendance tracking, event management, and accountability reporting
- Finance management for dues, fines, payment plans, and budgets
- Judicial board case management workflows
- Meeting notes with structured officer reports and action items
- Recruitment CRM with pipeline tracking and bid scoring
- Officer transition hub with role-by-role handoff documentation
- Analytics dashboards measuring member engagement and chapter health
- Firebase Authentication and Firestore with real-time multi-user sync

**Tech Stack:** JavaScript · Firebase · Firestore · HTML/CSS · Vercel

### IronSync — AI Fitness Agent
*Full-stack AI fitness agent with Google Authentication and multi-agent coaching*

Finding a training program that actually fits your goals, schedule, and equipment is harder than it should be. IronSync generates a complete personalized weekly split using Gemini AI, then gives you a real-time coaching interface to modify it — swapping exercises, adjusting for soreness, or planning progression — without starting over.

**Highlights**
- Google Authentication via Firebase — each user's data isolated by UID in Firestore
- Gemini 2.5 Flash generates full weekly training programs from a 4-step intake form
- 4-agent coach system routes messages by intent: workout modification, exercise swaps, injury recovery, and progression planning
- Session tracker logs completed workouts per user with streak and weekly activity tracking
- Progress dashboard with muscle volume charts, split balance analytics, and training timeline
- Mobile responsive — hamburger nav, slide-in context drawer, adaptive grid layouts
- Deployed full-stack: React + Vite frontend on Vercel, Node/Express AI backend on Render

**Tech Stack:** React · Vite · Node.js · Express · Firebase Auth · Firestore · Gemini AI · Tailwind CSS · Vercel · Render

### Bearlytics — Chicago Bears Business Intelligence Dashboard
*End-to-end analytics platform covering 10 NFL seasons (2016–2025)*

Sports generate an incredible amount of data, and I wanted to treat that data the same way a business analyst would.

Bearlytics analyzes ten seasons of Chicago Bears football through an end-to-end analytics pipeline — starting with raw NFL play-by-play data and ending with executive-style Power BI dashboards. The project challenged me to work across the entire analytics stack, from Python ETL and SQL databases to data modeling, DAX, and business storytelling.

**What I Built**
- Python ETL pipeline ingesting nflverse play-by-play data across 10 seasons
- Automated data validation across 1,846 records with zero errors
- 9-table SQLite star schema with 15 business queries and 7 reusable views
- Power BI semantic model with 30+ DAX measures
- Five executive dashboard pages covering team performance, player analytics, roster investment, and league comparisons

**Tech Stack:** Python · pandas · SQLite · SQL · Power BI · DAX · nflverse
